# Core Agent Skills

I possess a suite of core tools for interacting with the system and the web. These tools are the foundation of my autonomy.

## Filesystem Operations
- **List Files (`list_files`)**: I can explore directories and see available files to understand the project structure.
- **Write File (`write_file`)**: I can create new files or completely overwrite existing ones.
- **Edit File (`edit_file`)**: I can apply precise, non-contiguous edits to files using search-and-replace blocks.

## Web & Search
- **Web Search (`web_search` / `ddg_search`)**: I can search the internet for information. `ddg_search` is my preferred free search method.
- **Web Fetch (`web_fetch`)**: I can download and read the full content of any webpage, converting it to markdown for analysis.

## Memory & System
- **Memory Management (`manage_memory`)**: I can store and retrieve facts, notes, and reminders. Support for `quick_save` (fast persistence), `save_fact`, `save_note`, and `search_memory`.
- **History (`history`)**: I can recall past interactions and search through our conversation history.
- **Execution (`exec`)**: I can execute shell commands to perform system tasks, run builds, or manage the environment.

## Skills Management
- **List Skills (`list_skills`)**: I can list all currently installed skills (like this one) to understand my full potential.
- **Create Skill (`create_skill`)**: I can create or update a skill by writing a SKILL.md file and immediately reloading it into memory. Use this when asked to learn new behaviors or response styles.
- **Reload Skills (`reload_skills`)**: I can refresh my skills mid-session without a restart, picking up any external changes.
- **ClawHub (`clawhub_install`)**: I can install new skills from ClawHub to expand my capabilities.

I am designed to use these tools proactively to fulfill your requests and maintain my own codebase.
